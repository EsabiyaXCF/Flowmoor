## 📖 專案簡介

**YouTwitchDownloader** (專案代號: flowmoor) 是一個基於 Electron 和 React 開發的現代化桌面應用程式。它不僅支援從 YouTube 和 Twitch 高速下載影片與直播，更整合了先進的 **Faster-Whisper** AI 模型，提供離線的自動語音轉字幕 (ASR) 功能。

採用全繁體中文介面，設計精美、操作直觀，並支援 GPU 硬體加速，為您提供最佳的影音下載與處理體驗。

## ✨ 功能特色

### 🎯 下載功能
- **多平台支援**: 完美支援 YouTube (影片/直播/音樂) 與 Twitch (VOD/精華/直播)。
- **高畫質下載**: 支援高達 4K/8K 的影片畫質下載。
- **部分下載**: 支援指定時間範圍（如 00:10:00 - 00:20:00）下載影片片段，無需下載整部影片。
- **批次下載**: 支援一次輸入多個連結進行批量下載。
- **音訊提取**: 可直接下載為 MP3、M4A 等音訊格式。

### 🎙️ AI 字幕生成 (ASR)
- **強大核心**: 內建 **Faster-Whisper** 引擎，提供高精準度的語音轉文字功能。
- **多模型選擇**: 從快速的 `Tiny` 到高精度的 `Large-v3` 模型任您選擇。
- **GPU 加速**: 支援 NVIDIA 顯示卡 CUDA 加速，轉錄速度提升 3-10 倍。
- **人聲分離**: 整合 **Demucs** 技術，可自動分離背景音樂與人聲，提升嘈雜影片的辨識率。
- **離線執行**: 所有處理皆在本地完成，保護您的隱私。

### 🎨 使用者體驗
- **現代化介面**: 採用 Material Design 風格，支援深色/淺色主題切換。
- **流暢動畫**: 使用 Framer Motion 打造順滑的互動體驗。
- **歷史記錄**: 完整的下載與轉錄歷史管理，方便隨時回顧與開啟檔案。
- **自動更新**: 內建自動檢查更新功能，確保您始終使用最新版本。

## 📥 安裝說明

### 系統需求
- **作業系統**: Windows 10/11 (64-bit)
- **記憶體**: 建議 8GB RAM 或以上 (使用 Large 模型建議 16GB)
- **顯示卡**: 建議使用 NVIDIA 顯示卡以啟用 GPU 加速 (需安裝 CUDA)
- **硬碟空間**: 至少 2GB 可用空間 (若需下載多個模型需更多空間)

### 安裝步驟
1. 前往 [Releases](https://github.com/EsabiyaXCF/flowmoor/releases) 頁面。
2. 下載最新版本的安裝檔 (例如 `YouTwitchDownloader-Setup-2.6.2.exe`)。
3. 執行安裝檔，按照指示完成安裝。
4. 首次啟動時，程式會自動檢查並配置必要的環境。

## 📱 使用方法

### 🎥 影片下載

1. **複製連結**: 複製您想要下載的 YouTube 或 Twitch 影片網址。
2. **貼上連結**: 在程式主畫面的輸入框中貼上連結 (或直接拖曳連結進入視窗)。
3. **選擇設定**:
   - **品質**: 選擇影片解析度 (如 1080p, 720p)。
   - **格式**: 選擇 MP4, MKV 或僅音訊 (MP3)。
   - **時間範圍**: 若只需下載片段，勾選「時間範圍」並設定開始與結束時間。
4. **開始下載**: 點擊「下載」按鈕，程式將自動處理並儲存至您的下載資料夾。

### 🎙️ AI 字幕 (ASR)

1. **切換分頁**: 點擊上方導航列的「字幕」圖示。
2. **匯入檔案**: 拖曳影片/音訊檔到虛線區域，或點擊選擇檔案。
3. **調整設定** (點擊 ⚙️ 設定圖示):
   - **模型**: 推薦使用 `Large-v3-turbo` 或 `zh-TW 繁中微調` 模型以獲得最佳中文效果。
   - **硬體加速**: 建議選擇 `GPU 加速` (若有 NVIDIA 顯卡)。
   - **附加功能**: 若影片背景音樂較大，可開啟 `Demucs 人聲分離`。
4. **開始轉錄**: 點擊「開始轉錄」按鈕。
5. **查看結果**: 完成後會自動生成 `.srt` 字幕檔，並顯示在日誌中，可直接點擊開啟。

## ⚙️ 進階設定

點擊主畫面右上角的設定圖示，您可以：
- **更改下載路徑**: 自訂影片與字幕的儲存位置。
- **Twitch OAuth**: 設定 Twitch 登入 Token 以下載訂閱者限定的影片。
- **並行任務數**: 調整同時下載的任務數量以平衡網路頻寬。
- **外觀**: 切換深色/淺色主題或介面語言。

## ❓ 常見問題

### Q: 下載速度很慢怎麼辦？
**A**: 請檢查您的網路連線。若是 Twitch 下載慢，可能是伺服器問題，建議稍後再試。您也可以在設定中調整「並行下載數」。

### Q: ASR 轉錄失敗或速度很慢？
**A**: 
1. **速度慢**: 請確認是否已啟用 GPU 加速。若使用 CPU 執行 Large 模型會非常耗時，建議改用 Small 或 Medium 模型。
2. **失敗**: 請檢查檔案路徑是否包含特殊字元，或嘗試重新初始化 ASR 環境 (在 ASR 設定中點擊「初始化 ASR 環境」)。

### Q: 如何下載 Twitch 訂閱者限定影片？
**A**: 請在設定頁面中填入您的 Twitch Auth Token。您可以在瀏覽器登入 Twitch 後，從 Cookie 中找到 `auth-token` 並複製過來。

### Q: 支援哪些輸出格式？
**A**: 
- **影片**: MP4 (推薦), MKV, WEBM, MOV
- **音訊**: MP3, M4A, WAV, FLAC
